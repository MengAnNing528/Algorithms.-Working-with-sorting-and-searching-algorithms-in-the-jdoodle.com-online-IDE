# Algorithms.-Working-with-sorting-and-searching-algorithms-in-the-jdoodle.com-online-IDE

Краткое описание алгоритмов:

Блочная сортировка - распределяет элементы по "корзинам", сортирует каждую корзину отдельно, затем объединяет
Блинная сортировка - сортирует переворачиванием частей массива (как блины на сковороде)
Сортировка бусинами - визуализирует сортировку как падение бусин под действием гравитации
Поиск скачками - прыгает с фиксированным шагом, затем делает линейный поиск в блоке
Экспоненциальный поиск - экспоненциально увеличивает диапазон, затем использует бинарный поиск
Тернарный поиск - делит массив на три части и рекурсивно ищет в нужной части
